#  Multi-Tenant SaaS Dashboard

A full-stack, multi-tenant SaaS dashboard built with **React**, **Node.js**, and **MongoDB**, featuring dynamic theming and per-tenant data isolation.

---

##  Features
- Multi-tenant architecture with tenant-level data isolation
- Dynamic theming (light/dark/custom)
- Role-based authentication
- Real-time analytics
- REST + WebSocket support

---

## Tech Stack
**Frontend:** React, Tailwind CSS  
**Backend:** Node.js, Express.js  
**Database:** MongoDB (with tenant-based collections)

---

##  Installation

```bash
# Clone repo
git clone https://github.com/<your-username>/multi-tenant-saas-dashboard.git

# Backend
cd backend
npm install
npm start

# Frontend
cd ../frontend
npm install
npm start
